# surfanalytics-v2

## Git commands

1. Clone the code from repo

```bash
git clone https://github.com/thefilet-tb/surfanalytics-v2.git
```

2. Check the status of git repo

```bash
git status
```

3. Create new branch

```bash
git checkout -b
```

```bash
git fetch
git pull
```

5. Checklist of branches

```bash
git branch -a
```

6. Checkout main

```bash
git checkout main
```

7. Index new and modified files
    
```bash
git add .
```  

```bash
git commit -m "comment"
```
